PRIVATE HELPER — _pivot_cnv:
  Signature: def _pivot_cnv(parquet_path: pathlib.Path) -> pl.DataFrame
  Implementation:
    1. df = pl.read_parquet(parquet_path)
    2. Filter to canonical chromosomes only: df = df.filter(pl.col("chromosome").is_in(list(HG38_CENTROMERES.keys())))
       (This drops chrM, chrUn_*, and any non-standard chromosome names — avoids null chr_arm values)
    3. Build centromere lookup: cent_df = pl.DataFrame({"chromosome": list(HG38_CENTROMERES.keys()), "centromere": list(HG38_CENTROMERES.values())})
    4. Join and assign arm:
       df = (
           df.join(cent_df, on="chromosome", how="left")
           .with_columns([
               (
                   pl.col("chromosome").str.replace("chr", "")
                   + pl.when(((pl.col("start") + pl.col("end")) / 2) < pl.col("centromere"))
                   .then(pl.lit("p"))
                   .otherwise(pl.lit("q"))
               ).alias("chr_arm")
           ])
       )
    5. arm_means = df.group_by(["patient_id", "chr_arm"]).agg(pl.col("copy_number").mean())
    6. return arm_means.pivot(on="chr_arm", index="patient_id", values="copy_number", aggregate_function="first")
  NOTE: arm column names become "1p", "1q", "2p", etc. — the str.replace("chr","") strips the prefix, then "p" or "q" is appended.

  Signature: def _pivot_methylation(parquet_path: pathlib.Path, top_probes: list[str]) -> pl.DataFrame

    NOTE: aggregate_function="mean" handles null beta_values correctly (nulls are ignored in mean)

    4. Per-cohort loop:
         cohort_frames = []
